## 可控制变量

| 变量                     | 含义             |
| ------------------------ | ---------------- |
| --close-icon-bg-color    | 关闭图标背景颜色 |
| --close-icon-line-color  | 关闭图标线条颜色 |
| --close-icon-size        | 关闭图标容器大小 |
| --close-icon-line-width  | 关闭图标线条宽度 |
| --close-icon-line-height | 关闭图标线条高度 |
